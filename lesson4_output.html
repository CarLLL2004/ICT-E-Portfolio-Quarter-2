Lesson 4 - CSS Rules

What I Learned

CSS Rule
A CSS rule has two main parts: selector and one or more style declarations.
Inline styles are written inside the tag, while document-level and external style sheets use selectors and declarations separately.

Style Properties
Properties are grouped into six:
1. Fonts
2. Colors and backgrounds
3. Text
4. Boxes and layout
5. Lists
6. Tag classifications

Classes
A class is a user-defined selector used to apply style rules to specific elements.
It can control individual element formatting and is reusable across multiple elements.
Classes are defined by a dot and can be written with or without an element name.
A class must be called in the HTML file using the class attribute to apply its style.
One element can have more than one class, and one class can be applied to many elements.
Classes allow consistency and flexibility.

Class Selector
Element + class selectors apply styles only when both the element and class match.
Class selectors can be reused across elements, while element + class selectors are more specific.
.classname ‚Üí applies to any element with that class
element.classname ‚Üí applies only to that specific element type

External and Internal CSS
Internal CSS uses style rules inside the HTML file, while external CSS uses a linked stylesheet.
Classes can be applied to many elements for a consistent design across the page.

Grouping Selectors
Multiple selectors can be combined using commas to make the code shorter and cleaner.
This is useful for applying the same style to different elements without repeating rules.
