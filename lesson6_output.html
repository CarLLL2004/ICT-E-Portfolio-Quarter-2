Lesson 6 - Dimensions

What I Learned

Setting Dimensions
CSS can be used to manage dimensions such as visibility, width or height of HTML elements.

Visibility Property
Indicates if an element is visible or hidden.
visibility: visible ‚Äì default value, the element is visible.
visibility: hidden ‚Äì the element is hidden but still takes up space.
visibility: collapse ‚Äì used for table rows, columns, and flex items. Acts like display:none.

Dimension Properties
width ‚Äì sets width of the element.
height ‚Äì sets height of the element.
line-height ‚Äì sets line spacing.
max-height ‚Äì sets maximum height.
min-height ‚Äì sets minimum height.
max-width ‚Äì sets maximum width.
min-width ‚Äì sets minimum width.

HTML and CSS Example - Height & Visibility
<!DOCTYPE html>
<html>

body {background-color:#DAF02D;}

</html>

HTML and CSS Example - Width & Visibility
#myDIV {width:300px; background-color:#33EBC0;}
span{background:lightblue; visibility:hidden;}

HTML and CSS Example - With Class
.NoDimensions {background-color:pink; font-family:Times; font-size:30px;}
.HasDimensions {background-color:yellow; font-family:Arial; font-size:20px; width:50%;}
.Hidden {background-color:lightblue; font-family:Arial; font-size:12px; visibility:hidden;}

Pseudo-Classes and Links
Pseudo-classes are defined by a colon (:) and are not user-defined. They apply to specific HTML elements like links.
Four Anchor Pseudo-classes:
:link ‚Äì unvisited link
:visited ‚Äì visited link
:hover ‚Äì mouse hover
:active ‚Äì when clicked

LVHA Order: Link, Visited, Hover, Active

Pseudo-Class Example
a {font-family:Arial; font-size:15px; font-weight:bold; color:red; line-height:22px;}
a:link {color:red;}
a:visited {color:black;}
a:hover {color:green;}
a:active {color:blue;}

Practical Application:
Dimensions help control the size and space of elements on a webpage. Pseudo-classes make pages interactive, especially links.
