Lesson 5 - CSS Selectors

What I Learned

CSS selectors are used to find or select the HTML elements that you want to style.
There are 6 types of CSS selectors:
1. Class selector
2. Element selector or Tag selector
3. ID selector
4. Universal selector
5. Group selector
6. Attribute selector

Element Selector
Selects HTML elements like p, div, h1 and applies CSS to them.

ID Selector
Styles a single, unique element using an id and is defined by a #.
It is normally used only once on a page and must be unique.

Universal Selector
Selects every single HTML element on the page.
Useful for applying general styles like resetting margins and padding.

Group Selector
Allows you to select multiple elements and apply the same style to all of them.
This helps make the code cleaner and shorter.

Attribute Selector
Selects elements based on specific attribute values.
Useful for styling elements with certain attributes like class or type.
